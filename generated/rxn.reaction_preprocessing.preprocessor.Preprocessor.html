Creates a new instance of the Preprocessor class.

Parameters

  • df (DataFrame) – A pandas DataFrame containing the reaction SMARTS.

  • reaction_column_name (str) – The name of the DataFrame column containing the reaction SMARTS.

  • valid_column (str, default: '_rxn_valid') – The name of the column to write the validation results to (will be created if it doesn’t exist). Defaults to “_rxn_valid”.

  • valid_message_column (str, default: '_rxn_valid_messages') – The name of the column to write the validation messages to in verbose mode (will be created if it doesn’t exist). Defaults to “_rxn_valid_messages”.

  • fragment_bond (str, default: '.') – The token that represents fragment bonds in the reaction SMILES.

  • clean_data – Whether to run a simple pre-cleaning of the input data (naive check for valid SMARTS reactions based on the number of greater-thans in the string). Defaults to True.

filter(filter, verbose=False, filter_func=None, filter_func_verbose=None)[source]

Applies filter functions to the reaction. Alternatives for the default filter functions can be supplied. The default filters remove reactions containing molecules that could not be parse by rdkit’s MolFromSmiles.

Parameters

  • filter (MixedReactionFilter) – An instance of MixedReactionFilter to be applied to each reaction.

  • verbose (bool, default: False) – Whether or not to report the reasons for a reaction being deemed invalid. Defaults to False.

  • filter_func (Optional[Callable[[str, MixedReactionFilter], bool]], default: None) – A custom filter function. Defaults to None.

  • filter_func_verbose (Optional[Callable[[str, MixedReactionFilter], List[str]]], default: None) – A custom verbose filter function. Defaults to None.

reaction_standardization()[source]

Reaction standardization, including merging of reactants and reactants, removal of duplicates, sorting, etc.

Note that this standardization relies on the molecules in the reaction SMILES to be canonical - which is the case when this function is called as part of the full data processing pipeline.
